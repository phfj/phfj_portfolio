# ui-header: Header Visual Polish & Active Underline Marker

## What to build

Enhance the navigation header with a rich backdrop blur, a top-to-bottom background gradient wash, and a wider active page underline with a softened transition.

## Acceptance criteria

- [ ] Navigation header includes custom backdrop blur and top-to-bottom background gradient wash
- [ ] Active page link in the header is indicated by a smooth underline transition
- [ ] Header layout is responsive across screen sizes
- [ ] `npm run build` succeeds without type or lint errors
