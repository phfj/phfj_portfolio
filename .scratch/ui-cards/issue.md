# ui-cards: Cards Left-Border Hover Accent & Shadow Treatment

## What to build

Refactor Project and Post card components to animate a left-border strip on hover, lift their shadows, and format their metadata fields (categories, dates, reading times) using monospaced typography. Add a small-caps monospaced label (like "WORK" and "WRITING") above section titles. Highlight project categories in a slate-blue color.

## Acceptance criteria

- [ ] Project and Post cards animate a left-border strip on hover
- [ ] Cards lift slightly on hover with the custom shadow transition
- [ ] Project categories highlighted in slate-blue metadata color
- [ ] Project and post metadata (dates, categories, reading time) formatted in a monospaced font
- [ ] Small-caps monospaced labels (e.g., "WORK", "WRITING") render above section titles
- [ ] `npm run build` succeeds without type or lint errors

## Blocked by

- `ui-tokens`
