# ui-hero: Homepage Layout Clean-up & Editorial Typographic Hero

## What to build

Clean up the homepage layout by removing the newsletter subscribe form and GitHub live API integrations. Restructure the hero with a clean typographic layout, displaying the developer's name in a large, clean typeface, and a concise 2-3 sentence biography explaining their focus on systems engineering and human-centered design. Add a clean section-divider line below the hero.

## Acceptance criteria

- [ ] Homepage hero has the developer's name in a large, clean typeface
- [ ] Homepage hero displays the 2-3 sentence systems and human-centered design bio
- [ ] GitHub API integration widget and homepage newsletter form removed from homepage
- [ ] Layout includes a clean section-divider line separating the hero from subsequent sections
- [ ] `npm run build` succeeds without type or lint errors
