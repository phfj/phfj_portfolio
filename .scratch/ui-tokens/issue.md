# ui-tokens: Establish Design System Tokens & Accent Colors

## What to build

Establish the design system tokens for the portfolio website. Define slate blue metadata tokens (`--accent-blue`), custom warm-tinted card shadow tokens (`--shadow-card`, `--shadow-card-hover`), and monospaced font declarations in the global stylesheet. Update dark mode variables to ensure proper contrast accessibility for new colors.

## Acceptance criteria

- [ ] Slate blue token (`--accent-blue` or similar) established in CSS variables
- [ ] Warm-tinted card shadow (`--shadow-card`) and hover shadow (`--shadow-card-hover`) defined
- [ ] CSS utility classes or styles for monospaced elements created
- [ ] Dark mode variables updated to ensure proper contrast accessibility for new colors
- [ ] `npm run build` succeeds without type or lint errors

## Blocked by

None - can start immediately
