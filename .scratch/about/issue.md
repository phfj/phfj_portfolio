# About page

## What to build

A static `/about` page with the Author bio. No Sanity dependency — pure static
content. This is the simplest page on the site and can be built independently.

## Acceptance criteria

- [ ] `/about` route renders a static bio page
- [ ] Content includes: name, role, what the site is about, mission (self-education, curiosity, sparking interest in the field)
- [ ] Optional: links to social/github/etc.
- [ ] `npm run build` succeeds
- [ ] `npm run typecheck` exits 0
- [ ] `npm run lint` exits 0

## Blocked by

None — can start immediately.
