# ui-projects-intro: Projects Page Editorial Framing Component

## What to build

Introduce a hardcoded prose framing component (`FeaturedWorkIntro`) above the project filtering and list view. This component should display a warm, minimalist editorial introduction paragraph framing the collection of work.

## Acceptance criteria

- [ ] Static `FeaturedWorkIntro` prose framing component added to the projects page
- [ ] Editorial introduction paragraph correctly rendered above the project list and filters
- [ ] Component is styled to match the warm, minimalist design
- [ ] `npm run build` succeeds without type or lint errors

## Blocked by

None - can start immediately
